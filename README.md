English:
Favours alternation by using T + vowels, while the SFB being lower than other T + vowels layouts, at the price of higher SFS. Mainly inspired by the Ursnf and Poqtea layouts.

Simple English:
This keyboard layout makes you type with both hands. If the first letter is typed by one finger, you usually use another finger to type the second letter. I made this layout, when I was looking at the Ursnf and Poqtea layouts.

Deutsch:
Dieses Tastatur-Layout für Englisch fördert Handalternation und reduziert aufeinanderfolgende Gleichfingerbenutzung (German words, my ass). Die Inspiration kommt hauptsächlich von den Ursnf und Poqtea Layouts.
